# UI Cat a Fork of UI Kitten

Fork of [https://github.com/ui-cat/react-native-ui-cat](https://github.com/ui-cat/react-native-ui-cat).

## Migrate from UI Kitten to UI Cat

Replace `@ui-kitten/` -> `@ui-cat/` in your imports.  
Replace `@eva-design/` -> `@ui-cat/eva-design-` in your imports.

Packages:

- `@ui-kitten/components` -> `@ui-cat/components`
- `@ui-kitten/date-fns` -> `@ui-cat/date-fns`
- `@ui-kitten/eva-icons` -> `@ui-cat/eva-icons`
- `@ui-kitten/metro-config` -> `@ui-cat/metro-config`
- `@ui-kitten/moment` -> `@ui-cat/moment`
- `@eva-design/eva` -> `@ui-cat/eva-design-eva`
- `@eva-design/dss` -> `@ui-cat/eva-design-dss`
- `@eva-design/material` -> `@ui-cat/eva-design-material`
- `@eva-design/processor` -> `@ui-cat/eva-design-processor`
